name: Design philosophy
description: Site should stay punk and minimal — avoid overdesigning or adding unnecessary polish
type: project
---

The site is for a friend's band (Family Vision). The aesthetic is intentionally punk and minimal.

**Why:** The band wants a raw, stripped-down look. Overdesigning would go against the vibe.

**How to apply:** Resist adding visual flair, animations, extra UI components, or polished design touches. Keep changes functional and lean. When in doubt, do less.
